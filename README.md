# Library Management System

## Deskripsi
Program Python berbasis terminal untuk mengelola data buku perpustakaan sederhana.

## Fitur Program
- Tambah buku baru (Novel / Education)
- Tampilkan semua buku beserta statistik
- Cari buku berdasarkan judul
- Pinjam buku
- Kembalikan buku
- Simpan data ke file
- Auto-save otomatis setelah setiap perubahan data

## Cara Menjalankan Program
```
python main.py
```

## Contoh Format Data (books.txt)
```
Novel|Laskar Pelangi|Andrea Hirata|2005|Available
Education|Python Basic|John Doe|2023|Borrowed
```
